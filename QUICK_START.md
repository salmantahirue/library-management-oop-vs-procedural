# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Application
```bash
python app.py
```

### Step 3: Open Your Browser
Navigate to: **http://localhost:5000**

---

## 📋 What to Do Next

1. **Read the Problem Statement** - Understand what we're building
2. **Click "Procedural/Functional"** - Learn the traditional approach first
3. **Follow the Tutorial Steps** - Read each step carefully
4. **Test the System** - Use the interactive interface to add books, members, and perform operations
5. **Read the Limitations** - Understand what problems arise
6. **Click "Object-Oriented"** - See how OOP solves these problems
7. **Compare Both** - Notice the differences in code organization and data protection

---

## 💡 Pro Tips

- Start with **Procedural** approach to understand the limitations
- Try to break things - see what happens when you enter invalid data
- Compare the code examples side by side
- Read the code files directly: `procedural_approach.py` and `oop_approach.py`
